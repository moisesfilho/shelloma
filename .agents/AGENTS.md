# Regras do Projeto Shelloma

## Regras de Execução de Comandos Sudo
- **Comandos Sudo**: Nunca execute comandos precedidos por `sudo` diretamente.
- Quando for necessário utilizar privilégios de administrador (ex: `sudo apt install`, `sudo dpkg -i`), apresente o comando exato em um bloco de código de terminal no chat e solicite que o usuário o execute manualmente.
- Aguarde a resposta/confirmação do usuário sobre o resultado da execução antes de continuar com as etapas dependentes.
